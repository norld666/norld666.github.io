【甘々ふたなり】生えちゃうヒーラーの妹と、相互射精あまあまえっち





ふたなりの妹と、相思相愛いちゃらぶえっち。

現実と少し離れた世界。魔力を持つ妹とふたりきりで過ごす日常。
とある日のこと、妹の身体が「ふたなり」になってしまったと告げられます。
性欲をたくさん解消してあげれば、一応女の子の身体に戻るようですが…
頻繁に発情してしまうので、毎日のようにえっちを求めて来ます。

ふたなりおち〇ぽをお世話したり、逆に挿入されちゃったり…
もちろん、女の子としても愛してあげるのがアナタの役目です。

■作品コンセプト

「ふたなり妹+あまあま」がコンセプト。

普通の女の子としてのえっちも、ふたなりを受け止めるえっちも。
…もちろん、妹に犯されるシチュエーションもあります。

ふたなりの女の子を全部愛してあげたい…!という想いの作品です。

■作品内容

01:ハメっぱなしの寝起き即えっち　(※ふたなり無し)　(19:57)　
▶【騎乗位 / キスハメ / 搾精まどろみ朝えっち / べろちゅー中出し】

妹と一緒に目覚めるいつもの朝。
前日にたくさんえっちをしたおかげで、一時的に女の子の身体に戻った様子でしたが…
ふたなりおち〇ぽが無くても関係ありません。
夜から繋がりっぱなしの騎乗位体勢で、まどろみながら搾り取られちゃいます。
━━━━━━━
02:妹と一緒におち〇ぽオナニー　(20:18)
▶【逆手コキ / べろちゅー / 兜合わせコキ / 一緒にしこしこ射精】

数日後。身体が安定してきたとの事で、一人で外出していた妹でしたが…
少しえっちをサボっただけで、再びふたなりおち〇ぽが生えてきてしまいました。

身動きが取れない妹の元に駆けつけたアナタ。
そのまま宿屋へと駆け込み、発情した妹とおち〇ぽくっ付け同時オナニー。
二人でしこしこ握り合いながら、一緒にたくさん気持ちよくなります。
━━━━━━━
03:寝バック逆アナルでえっちなマッサージ　(25:53)
▶【寝バック逆アナル / 敬語でマッサージごっこ / 耳元囁き / 逆中出し】

宿屋で過ごす日の夜、何度か射精をしても発情が治まらない妹。
背中からぎゅっと抱きつかれて、妹の柔らかい身体を押し付けられながらえっちなマッサージごっこ。
そうこうしているうちに、勃起したふたなりおち〇ぽを後ろから挿入されて…

全身をゆっくり攻められる逆アナルえっちで、お腹の中に妹ザーメンをたくさん注がれちゃいます。
━━━━━━━
04:密着いちゃいちゃべろちゅー正常位　(28:56)
▶【正常位 / キスハメ / 唾液交換 / だいしゅきホールド /
べろちゅー中出し / 妹ザーメンお漏らし / 一緒にとろとろ射精】

とある書物から、ふたなりに関する情報を手に入れたという妹。
「魔力が多い女の子は、その力が原因でふたなりの影響が出てしまう」といいます。
解決方法は今まで「たくさんえっちをして魔力を放出する」のが一番だと分かりました。

アナタへの想いが更に強くなった妹から、正常位でおま〇こえっちのおねだり。
隙間なく抱き合い、ふたなりおち〇ぽを刺激しながらのとろとろ甘々愛情キスハメ。
妹の気持ちいい場所を両方愛してあげながら、いちゃらぶべろちゅー射精で幸せ同時絶頂です。
━━━━━━━
05-1:発情妹おち〇ぽに襲われ逆アナル　(23:05)
▶【正常位逆アナル / 発情ピストン / オホ声 / べろちゅー / 逆中出し】

しばらく経って身体も落ち着き、聖職者としての活動も再開した妹。
ま両者を正式なパートナーとして認め、家族以上の関係になると約束しました。

おち〇ぽを「無くす」のではなく、魔力のコントロールで「自在にできる」すべを身につけた妹ですが…
性欲処理の必要は無くてもえっちなことは欠かしません。
「ふたなりの妹」を求めるアナタを見て、愛しさのあまり暴走してしまいます。

いつもより激しく乱暴な、本能をむき出しのあまあま逆アナルレ〇プ。
途中で軽くイかされながら、恥ずかしい声を出して快楽をむさぼる交尾セックスです。
━━━━━━━
05-2:抜かずのべろちゅー2回戦　(44:22)
▶【抜かずの2回戦 / 抱きしめ逆アナル / オホ声 / むさぼりべろちゅー
女の子ザーメン種付け / 一緒にべろちゅー射精】

引き続き前から思い切り抱きつかれて、密着逆アナル正常位。
妹の柔らかくて大きな胸に甘えたり、キスでお口を犯されながらのふたなりおち〇ぽピストン。

中も外もトロトロになりながら、気持ちいいことだけを求めるふたり。
とびきりの愛情をぶつけ合う濃厚セックスで、最後は女の子ザーメンをべろちゅー種付け射精されちゃいます。
